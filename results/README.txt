Contains saved artifacts for paper figures and reproducibility.

PER MODEL (mbert_ / chinese_bert_ prefix):
  *_similarity_matrix.npy  — full pairwise cosine similarity matrix (6306 x 6306)
  *_intra_pairs.npy        — intra-radical cosine similarity values (same radical pairs)
  *_inter_pairs.npy        — inter-radical cosine similarity values (cross radical pairs)
  *_euclid_intra_pairs.npy — intra-radical Euclidean similarity [1/(1+d)] values
  *_euclid_inter_pairs.npy — inter-radical Euclidean similarity [1/(1+d)] values
  *_bootstrap.npy          — bootstrap distribution of cosine Δ(intra−inter) (1000 resamples)
  *_euclid_bootstrap.npy   — bootstrap distribution of Euclidean Δ (1000 resamples)
  *_permutation_scores.npy — permutation test null distribution (1000 shuffles)
  *_rad_sizes.npy          — radical group sizes (68 values)
  *_rad_cohesions.npy      — per-radical mean intra-similarity (68 values)

SUMMARY:
  main_results.csv         — all statistics: cosine + Euclidean (intra, inter, p, d, CI, perm_p, size bias)

Generated by: scripts/radical_cohesion_test.py
Dataset: data/radical_dataset.csv (6,306 chars, 68 radicals, Unihan + Chinese-BERT vocab filter)
